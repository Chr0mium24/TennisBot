# 20cm / 25cm / 30cm baseline 深度误差分析 - 2026-07-13

## 1. 分析目标

本文只分析双目三角化的深度误差，不分析完整落点预测、不分析底盘接球闭环。

要比较的是三个 baseline：

| 方案 | baseline |
|---|---:|
| B20 | 0.20 m |
| B25 | 0.25 m |
| B30 | 0.30 m |

在本文中，相机焦距、YOLO 中心偏移、标定残差都固定为当前实测值。最终深度误差只随两个变量变化：

- 目标距离 `Z`
- baseline `B`

## 2. 误差来源

双目深度由视差决定：

```text
Z = f * B / d
```

深度误差近似为：

```text
dZ = Z^2 / (f * B) * dd
```

其中：

- `dZ` 是深度误差；
- `Z` 是目标距离；
- `f` 是像素焦距；
- `B` 是 baseline；
- `dd` 是视差误差。

这个式子说明两个关系：

1. 距离越远，深度误差按 `Z^2` 放大；
2. baseline 越大，深度误差按 `1 / B` 下降。

因此，在其他条件固定时，baseline 从 `20cm` 增加到 `25cm` 或 `30cm`，只能按比例降低深度误差，不能改变远距离误差按平方增长的趋势。

## 3. 用于计算的固定输入

本文只保留影响深度误差计算的原始数据。

### 3.1 标定输入

当前标定分辨率是 `1280x720`，YOLO 图像是 `3840x2160`。按相同视场缩放到 4K 坐标后：

| 项目 | 数值 |
|---|---:|
| 4K 等效像素焦距 `f` | 1266.934 px |
| 4K 等效 stereo RMS | 0.636 px |
| 当前 stereo RMS 原始值 | 0.2121 px |
| 当前 epipolar RMS 原始值 | 0.2568 px |

这里保留 stereo RMS 和 epipolar RMS，是为了说明当前几何标定质量在像素级范围内。深度误差计算中实际使用的是 `f = 1266.934 px` 和 `stereo RMS = 0.636 px`。

### 3.2 YOLO 输入

YOLO 偏移只统计已经正确检测到的框，即 `IoU >= 0.5` 的匹配框。不把漏检、错检、明显偏移框计入这里的中心偏移。

| 项目 | 数值 |
|---|---:|
| 验证图像 | 975 |
| GT 框 | 792 |
| YOLO 预测框 | 546 |
| IoU >= 0.5 的匹配框 | 377 |

匹配框中心偏移如下：

| 指标 | p50 | p95 |
|---|---:|---:|
| center error | 1.431 px | 5.673 px |
| abs dx | 0.759 px | 3.606 px |
| abs dy | 0.889 px | 4.243 px |

深度误差主要受 x 方向影响，因此后续使用 `abs dx` 计算视差误差。

## 4. 从 YOLO 偏移得到视差误差

左右相机的 x 方向中心误差合成为视差误差：

```text
dd_yolo = sqrt(2) * abs_dx
```

再叠加标定残差：

```text
dd = sqrt(dd_yolo^2 + stereo_rms^2)
```

得到：

| 场景 | abs dx | YOLO 视差误差 | 合成视差误差 `dd` |
|---|---:|---:|---:|
| p50 | 0.759 px | 1.073 px | 1.247 px |
| p95 | 3.606 px | 5.100 px | 5.139 px |

后续分别计算 p50 和 p95 两种情况。

## 5. 固定输入后的深度误差公式

把 `f = 1266.934 px` 和上面的 `dd` 代入后：

```text
dZ_p50 = 0.0009846 * Z^2 / B
```

```text
dZ_p95 = 0.0040565 * Z^2 / B
```

单位：

- `Z`: m
- `B`: m
- `dZ`: m

到这里，除了目标距离 `Z` 和 baseline `B`，其他变量都已经固定。

## 6. p50 深度误差

p50 表示在已经正确检测到目标时，YOLO 中心偏移的中位数情况。

| baseline | 5m | 10m | 15m | 20m | 23.77m |
|---|---:|---:|---:|---:|---:|
| 20cm | 0.123 | 0.492 | 1.108 | 1.969 | 2.782 |
| 25cm | 0.098 | 0.394 | 0.886 | 1.575 | 2.225 |
| 30cm | 0.082 | 0.328 | 0.738 | 1.313 | 1.854 |

## 9. baseline 比例关系

由于深度误差中的 baseline 项是 `1 / B`，所以不同 baseline 的误差比例可以直接计算：

| baseline 变化 | 深度误差比例 | 深度误差下降 |
|---|---:|---:|
| 20cm -> 25cm | 0.800x | 20.0% |
| 20cm -> 30cm | 0.667x | 33.3% |
| 25cm -> 30cm | 0.833x | 16.7% |

这个比例对 p50 和 p95 都成立，因为 baseline 只在分母里，和视差误差分位数无关。
