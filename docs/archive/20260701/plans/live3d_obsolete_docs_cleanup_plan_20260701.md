# Live3D Obsolete Docs Cleanup Plan

## Scope

Remove documentation and archived report artifacts that describe the deleted
standalone Live3D acceptance command.

## Steps

1. Delete archived Live3D acceptance report files and their captured frame
   images.
2. Delete the previous removal-plan/result notes that only documented that
   command.
3. Remove stale command references from current docs and broader archive notes.
4. Verify no current docs or Live3D files reference the removed command.
